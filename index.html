GitHub 이미지 업로더

이미지를 GitHub 저장소에 업로드하고, 상세페이지·랜딩페이지·HTML에서 불러올 수 있는 이미지 URL을 자동 생성합니다.

1. 저장소 설정

이 페이지는 서버 없이 작동합니다. GitHub에 실제 업로드하려면 개인용 토큰이 필요합니다. 토큰은 이 페이지에 저장하지 않는 것을 권장합니다.
GitHub Pages를 켠 공개 저장소를 권장합니다.
권한은 해당 저장소의 Contents: Read and write만 주는 것을 권장합니다.

수동 URL 생성

GitHub에 직접 올린 파일명도 여기 넣으면 URL을 만들 수 있습니다. 한 줄에 하나씩 입력하세요.

2. 이미지 업로드

이미지를 끌어오거나 클릭해서 선택

JPG, PNG, GIF, WebP, SVG 업로드 가능 / 여러 장 가능

아직 선택된 이미지가 없습니다.
